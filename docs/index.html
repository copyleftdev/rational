How this was built

Two engines. The model never does the arithmetic.

Cost models are exactly where floating-point lies and language models fumble — tiny per-token prices, enormous token counts, compounding rates. So the labor is split.

01Research agents gather

Live web search returns current prices and benchmarks with source URLs and observation dates. Half of 2026's models are already legacy — memory won't do.

02An exact kernel computes

Every figure runs through agent-calc in exact rationals. The calculator below mirrors it with exact BigInt fractions — no float drift, in your browser.

03You re-run it

Every chart reacts to your inputs; the whole repo reproduces with make all. Same inputs → bit-identical numbers. That's the argument.
